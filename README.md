# Ruta 38 Repuestos — Sistema POS

Sistema de punto de venta para repuestos de autos. Funciona 100% en el navegador, sin servidor ni instalación.

## Cómo usar
Abrir `index.html` directamente en el navegador (Chrome o Edge recomendado).

## Funciones
- Caja / Ventas con escáner de código de barras
- Inventario de productos
- Historial de ventas
- Reportes e impresión de tickets (impresora POS 80mm)

## Datos
Los datos se guardan en `localStorage` del navegador. No requiere conexión a internet.
